# 📚 Guia Completo - Site de Estudos Inteligente

Bem-vindo ao seu site de estudos com sistema inteligente de organização de conhecimento! Este guia explica todas as funcionalidades.

## 🎯 O Que É Este Site?

Um site de estudos que:
- ✅ Organiza automaticamente tópicos por dificuldade
- ✅ Mostra dependências entre conceitos
- ✅ Gera conteúdo automaticamente
- ✅ Visualiza conhecimento em pirâmide
- ✅ Permite pesquisar e adicionar novas matérias

## 🚀 Começando

### 1. Abrir o Site
1. Abra o arquivo `index.html` em um navegador
2. Você verá a página inicial com as 3 seções principais
3. Navegue usando o menu no topo

### 2. Estrutura Principal

```

```

## 🔍 Funcionalidade 1: Pesquisar Matéria

### O Que Faz?
Permite buscar qualquer tópico e o site organiza automaticamente mostrando:
- Nível de dificuldade
- Pré-requisitos necessários
- Tópicos relacionados
- Exemplos práticos

### Como Usar?

1. **Clique em "🔍 Pesquisar Matéria"** no menu
2. **Digite um tópico** (ex: "Equação de 2º grau", "Subtração", "Movimento")
3. **Veja os resultados** organizados por dificuldade
4. **Clique em um resultado** para ver detalhes
5. **Clique em "✨ Criar Aula Automaticamente"** para gerar a aula

### Exemplo Prático

```
Você digita: "Equação de 2º grau"
↓
O site encontra e mostra:
- Nível 9 (Especialista)
- Pré-requisitos: Equação 1º grau, Radiciação, Potenciação
- Relacionados: Funções, Sistemas de Equações
- Exemplos: x² - 5x + 6 = 0
↓
Você clica "Criar Aula"
↓
Uma aula completa é gerada com explicações, exemplos e exercícios!
```

## 🔺 Funcionalidade 2: Pirâmide de Conhecimento

### O Que Faz?
Visualiza todos os 24 tópicos em uma pirâmide mostrando a progressão do conhecimento:

```
        🟣 Especialista (Nível 9-10)
       🔴 Avançado (Nível 6-8)
      🟠 Intermediário (Nível 3-5)
    🟢 Iniciante (Nível 1-2)
```

### Como Usar?

1. **Clique em "🔺 Pirâmide"** no menu
2. **Veja a pirâmide visual** com 4 níveis
3. **Clique em um nível** para filtrar tópicos
4. **Veja o caminho recomendado** de aprendizado
5. **Clique em um tópico** para mais detalhes

### Cores da Pirâmide

- 🟢 **Verde** (Nível 1-2): Iniciante - Conceitos fundamentais
- 🟠 **Laranja** (Nível 3-5): Intermediário - Operações e aplicações
- 🔴 **Vermelho** (Nível 6-8): Avançado - Conceitos complexos
- 🟣 **Roxo** (Nível 9-10): Especialista - Tópicos avançados

## 📊 Funcionalidade 3: Mapa de Dependências

### O Que Faz?
Mostra quais tópicos você precisa aprender antes de um outro.

### Exemplo

```

↑ Expressões Numéricas
↑ Radiciação
↑ Potenciação

Tópicos Relacionados (pode aprender depois):

```

## 🎓 Funcionalidade 4: Geração Automática de Aulas

### O Que Faz?
Quando você seleciona um tópico, o site gera automaticamente:

1. **Explicações** - Conceitos claros e simples
2. **Exemplos Práticos** - Situações do dia a dia
3. **Exercícios** - Perguntas para testar conhecimento
4. **Dicas** - Orientações para melhor aprendizado

### Exemplo de Aula Gerada

```

📖 EXPLICAÇÃO
"Números naturais são aqueles que usamos para contar: 0, 1, 2, 3..."

💡 EXEMPLOS

- Idade: 15 anos

✅ EXERCÍCIOS
1. Qual é o primeiro número natural?
   a) 0
   b) 1
   c) -1
   d) Não existe
   
2. Números naturais incluem negativos?
   a) Sim
   b) Não
   c) Às vezes
   d) Não sei

💡 DICAS
- Este é um tópico fundamental
- Pratique os exercícios
- Revise regularmente
```

## 🔧 Personalizando o Site

### Mudar Cores

Edite `css/style.css`:

```css
:root {
    --cor-primaria: #2c3e50;      /* Azul escuro */
    --cor-secundaria: #3498db;    /* Azul claro */
    --cor-destaque: #e74c3c;      /* Vermelho */
    --cor-sucesso: #27ae60;       /* Verde */
}
```

### Adicionar Novos Tópicos

Edite `data/topicos.json` e adicione um novo objeto:

```json
{
    "id": "novo-topico",
    "nome": "Nome do Tópico",
    "categoria": "matematica-basica",
    "dificuldade": 5,
    "descricao": "Descrição breve",
    "dependencias": ["topico-1", "topico-2"],
    "relacionados": ["topico-3"],
    "conteudo": {
        "explicacao": "Explicação...",
        "exemplos": ["Exemplo 1", "Exemplo 2"],
        "exercicios": []
    }
}
```

### Adicionar Imagens

1. Salve a imagem em `pages/images/`
2. No HTML, adicione:

```html
<div class="container-imagem">
    <img src="../../images/nome-da-imagem.png" alt="Descrição">
    <figcaption>Descrição da imagem</figcaption>
</div>
```

## 🐛 Troubleshooting

### O site não abre
- Certifique-se de abrir `index.html` em um navegador
- Não abra como arquivo local (use um servidor local se possível)

### A pesquisa não funciona
- Verifique se `data/topicos.json` está no lugar correto
- Abra o console (F12) para ver erros

### As imagens não aparecem
- Verifique se o caminho está correto
- Use `../../images/nome.png` para aulas

### Os exercícios não funcionam
- Verifique se `js/script.js` está sendo carregado
- Abra o console (F12) para ver erros
