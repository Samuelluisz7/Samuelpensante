🤖 Como Obter Sua Chave de API Gemini

Passo a Passo:

  1. Acesse ai.google.dev
  2. Clique em "Get API Key"
  3. Clique em "Create API key in new project"
  4. Copie a chave gerada (começa com "AIza...")
  5. Cole a chave no campo abaixo
  6. Clique em "Salvar Chave"
  7. Clique em "Testar Conexão" para verificar
💡 Informação Importante:
Sua chave de API é salva localmente no seu navegador. Nunca é enviada para servidores externos. Você pode deletar a chave a qualquer momento.
