# ⚡ Setup Rápido - 5 Minutos

## 🎯 Objetivo
Colocar seu site online permanentemente no Netlify com domínio próprio.

## 📋 Checklist

### ✅ Pré-requisitos
- [ ] Ter este repositório Git pronto
- [ ] Ter uma conta de email

### ✅ Passo 1: GitHub (2 min)
- [ ] Criar conta em https://github.com
- [ ] Criar repositório `samuelpensante-estudos`
- [ ] Copiar link do repositório

### ✅ Passo 2: Upload para GitHub (1 min)
```bash
cd /home/ubuntu/site-estudos
git remote add origin SEU_LINK_AQUI
git branch -M main
git push -u origin main
```

### ✅ Passo 3: Netlify (2 min)
- [ ] Criar conta em https://netlify.com
- [ ] Conectar com GitHub
- [ ] Selecionar repositório
- [ ] Clicar "Deploy"
- [ ] Copiar link do site (ex: samuelpensante-estudos.netlify.app)

### ✅ Passo 4: Domínio (Opcional)
- [ ] Registrar em https://www.registro.br (R$ 30-50)
- [ ] Conectar ao Netlify
- [ ] Aguardar 24-48h

## 🔗 Links Importantes

| Serviço | Link | Tempo |
|---------|------|-------|
| GitHub | https://github.com | 1 min |
| Netlify | https://netlify.com | 1 min |
| Domínio | https://www.registro.br | 5 min |

## 🚀 Resultado

```
https://samuelpensante-estudos.netlify.app  ← Automático
https://samuelpensante.com                   ← Com domínio
```

## 💡 Dicas

1. **GitHub:** Use seu email real
2. **Netlify:** Conecte com GitHub (mais fácil)
3. **Domínio:** Registro.br é mais barato
4. **Segurança:** HTTPS automático no Netlify

## ⏱️ Tempo Total
- Sem domínio: **5 minutos**
- Com domínio: **5 minutos + 24-48h propagação**
