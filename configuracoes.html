🤖 Configuração da IA Gemini

Como obter sua chave de API Gemini?

  1. Acesse ai.google.dev
  2. Clique em "Get API Key"
  3. Crie um novo projeto ou selecione um existente
  4. Copie a chave gerada
  5. Cole a chave no campo abaixo

Nota: A chave é salva localmente no seu navegador. Nunca é enviada para servidores externos.

Sua chave de API será usada para gerar conteúdo com IA. Ver tutorial

📖 Integração com Wikipédia

A Wikipédia é integrada automaticamente! Quando você pesquisa um tópico, o site busca automaticamente na Wikipédia e combina com as respostas da IA Gemini para fornecer conteúdo mais rico e confiável.

✅ Wikipédia está sempre disponível e não requer configuração!

💾 Cache e Performance

O site salva automaticamente os resultados de pesquisas para melhorar a performance. Resultados em cache expiram após 1 hora.

🔒 Privacidade e Segurança

ℹ️ Informações Importantes:
  • Sua chave de API é salva localmente no navegador
  • Nunca é enviada para servidores de terceiros
  • Você pode limpar a chave a qualquer momento
  • As pesquisas são enviadas apenas para Google Gemini e Wikipédia
